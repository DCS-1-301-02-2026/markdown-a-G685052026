# 拓殖大学 情報工学科💻
#### ⚒️学科の概要と4つの専門分野
拓殖大学情報工学科では、高度情報化社会を支える「IT・AI・ネットワーク」の技術を基礎から応用まで総合的に学びます。カリキュラムは主に以下の4つの専門分野で構成されています。

##### 1.専門分野の構成
1. AI・データサイエンス： 人工知能の仕組みやビッグデータ解析を学ぶ

1. ソフトウェア・Web開発： Python や Java を用いたシステム構築

1. ネットワーク・セキュリティ： 安全な通信インフラの設計と防衛技術

1. 人間情報・メディア： CG、VR（仮想現実）、画像処理技術の研究

##### 2. 学年ごとの目標
- 1年次： プログラミングの基礎と、数学・物理などの理工学の土台作り

- 2年次： 応用的なプログラミングと、ネットワークやハードウェアの基礎学習

- 3年次：専門的な実験・演習各研究室への配属

- 4年次： 4年間の集大成となる「卒業研究」の実施と論文作成

# 卒業後の進路と主な就職先
| 職種 | 主な仕事内容 | 進路の例 |
|:---|:---:|---:|
| **システムエンジニア（SE）** | 企業の業務システムの設計・開発 | ITベンダー |
| **プログラマー** | Webアプリやスマホアプリの製造 | 開発会社 |
| **ネットワークエンジニア** | 通信インフラの構築・保守・管理 | 通信キャリア |
| *データサイエンティスト* | AIを活用したデータの分析と提案 | シンクタンク |
